# cordova-todo-list

## 
```bash
# 创建一个项目：
cordova create myDemo

# 进入myDemo目录下： 
cd myDemo

# 添加platform broswer： 
cordova platform add browser

# 启动myDemo： 
cordova run browser

# 运行相关平台的代码
cordova run <platform> // cordova run android
```
